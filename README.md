### World Map Explorer🌍
An interactive web app with:  
- A clickable world map.  
- Country details (population, area, etc.).  
- Top countries by GDP.  
- Landmark fetching via API.  

### How to Run🌍  
1. Clone the repo.  
2. Open `index.html` in a browser or use a local server.
